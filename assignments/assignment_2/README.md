# Northstar Planner App

## App overview and main features

Northstar Planner is a multi-screen student productivity app built with React Native and Expo. It uses a warm, product-like visual style, smooth motion, and a simple navigation flow that feels closer to a real mobile app than a classroom demo.

Main features:

- Home screen with animated intro content and quick access cards.
- Category menu screen with reusable feature cards.
- Stack-based detail screen with routed item data.
- Planner screen with a weekly timeline layout.
- Profile/settings screen with consistent design system styling.
- Animation demo screen with fade, scale, progress, and drag interactions.
- Bottom tab navigation combined with a stack detail flow.
- Responsive layout behavior for smaller and larger screens.

## Steps to install dependencies and run the app

From the assignment folder:

```bash
cd assignments/assignment_2
npx expo start
```

## Known issues or limitations

- All content is static sample data for the assignment.
- The animation demo is implemented with the React Native Animated API and PanResponder only, so it stays self-contained and backend-free.
