# Campus Companion App

## App overview and main features

Campus Companion is a small multi-screen React Native app for new students to quickly access key college information.

Main features:

- Home screen with quick navigation to all app sections.
- Contacts screen with important campus contacts shown using FlatList.
- Contact details screen with parameter passing from the contacts list.
- Schedule screen with a weekly class overview.
- Notice Board screen with announcements and a campus map link.
- Bottom Tab + Stack navigation combination.
- Dynamic light/dark theme toggle on Home.
- Responsive layout behavior for smaller and larger screens.

## Steps to install dependencies and run the app

From the assignment folder:

```bash
cd assignments/assignment_1
npx expo start
```

## Known issues or limitations

- Contact and timetable data are static sample data for this assignment.
- External map links require internet connectivity.
